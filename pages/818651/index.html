# 1.编译型/解释型

  • 编译型:在代码编写完成之后编译器将其变成另外一个文件教给你算计执行.

    代表语言:Java,c,c++ ,c#, Go...

  • 解释型:写完代码后交给解释器,解释器从上到下一行行代码执行:边解释边执行.(实时翻译)

    代表语言:python, PHP, JavaScript...

# 2.代码文档头文件

#!/usr/bin/env python     (解释器路径)

# -*- conding:utf-8 -*-   (默认解释器编码为utf-8)

  • .startswith() 判断是否以()开头,输出值为bool类型
